# Stone Paper Scissor Game

## Overview
The **Stone Paper Scissor Game** is a simple interactive web game built using **HTML, CSS, and JavaScript**.  
Players can choose between stone, paper, or scissors and compete against the computer. The game logic determines the winner instantly and updates the score dynamically.

This project demonstrates **JavaScript game logic, DOM manipulation, and responsive UI design**.

---

## Features

- Play Stone, Paper, Scissor against the computer
- Random computer choice generation
- Instant result display (Win / Lose / Draw)
- Score tracking
- Interactive and responsive UI

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
